# Audiobookshelf

Self-hosted audiobook and podcast server.

## Ports
| Port | Purpose |
|------|---------|
| 13378 | Web UI |

## Usage
docker compose up -d

## Notes
- Place audiobooks in /home/eloy/audiobookshelf/audiobooks
- Place podcasts in /home/eloy/audiobookshelf/podcasts
- Metadata and config persisted across container restarts
